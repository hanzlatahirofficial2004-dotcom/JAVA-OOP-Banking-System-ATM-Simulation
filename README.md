# 💻 Java Banking System – ATM Simulation  

A simple **Java-based Banking System (ATM Simulation)** built with **OOP principles**.  
This console app allows users to Deposit 🏦, Withdraw 💸, Check Balance 📊, and Calculate Interest 📈.
